# @artsy/auto-config

Artsy's [shared configuration](https://intuit.github.io/auto/pages/autorc.html#extending) for [auto](https://github.com/intuit/auto) a tool that enables PR releases based on labels.

## Contributing

The configuration is stored in the `package.json` under the `auto` key. Any updates should be made there.
